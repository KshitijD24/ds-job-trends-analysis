# Data Science Job Market Analysis

## Project Overview
Analysis of job posting trends, skill requirements, and hiring patterns in the data science field.

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn, Plotly
- spaCy for NLP
- Jupyter Notebook
